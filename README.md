# CA DMV Test Prep 🚗

Practice for the California DMV written knowledge test with Hebrew explanations.

**Live App**: [https://yoad.github.io/ca-dmv-prep/](https://yoad.github.io/ca-dmv-prep/)

## Features

- 🎓 **Learn Mode** — Detailed topic explanations in Hebrew
- 📚 **Study Mode** — 165 practice questions by category with instant feedback
- 📝 **Practice Test** — 46-question simulation (83% pass threshold)
- 📋 **Quick Reference** — Sortable tables: speeds, distances, curb colors, BAC limits
- 🔄 All distances in miles/feet (as in the real test) with km/m conversions in Hebrew explanations
- 📱 Responsive design — works on iPad, phone, and desktop
- 💾 Progress tracking via localStorage
